**Документация к приложению `word_pair_counter`**

---

### **Название**

`word_pair_counter` - Утилита командной строки для подсчёта пар слов в текстовом файле.

---

### **Обзор**

`word_pair_counter` — это простое приложение на языке Python, предназначенное для подсчёта количества раз, когда два заданных слова (без учёта регистра) встречаются в текстовом файле в пределах определённого максимального расстояния друг от друга (в количестве слов между ними).

---

### **Синтаксис**

```bash
python -m src.main <file> <word1> <word2> <max_dist>
```

---

### **Параметры**

| Параметр  | Обязательный | Описание |
| :-------- | :----------- | :------- |
| `<file>`  | Да           | Путь к текстовому файлу, в котором будет производиться поиск пар слов. |
| `<word1>` | Да           | Первое слово для поиска. |
| `<word2>` | Да           | Второе слово для поиска. |
| `<max_dist>` | Да        | Максимально допустимое расстояние (включительно) между `<word1>` и `<word2>`. Это целое число, не меньшее нуля. |

---

### **Описание**

Приложение читает указанный текстовый файл и анализирует его, разбивая на токены (слова). Оно ищет последовательности, где `<word1>` встречается, а затем, в пределах `max_dist` слов, встречается `<word2>`. Подсчитывается количество таких пар. Поиск не чувствителен к регистру. Слова разделяются любыми не-буквенными символами (например, пробелами, знаками препинания).

---

### **Возвращаемые значения**

Приложение выводит в стандартный поток вывода (`stdout`) одно целое число — количество найденных пар слов, соответствующих критериям. В случае ошибки (например, файл не найден или недопустимое значение `max_dist`) приложение выводит сообщение об ошибке в стандартный поток ошибок (`stderr`) и завершается с ненулевым кодом возврата.

---

### **Примеры**

**Пример 1:** Подсчёт пар слов "налоги" и "доходы" в файле `example.txt` с расстоянием не более 3 слов.

```bash
python -m src.main example.txt налоги доходы 3
```

Если `example.txt` содержит:

```
налоги на доходы физических лиц
```

Результатом будет:

```
1
```

**Пример 2:** Поиск пары слов в соседних позициях.

```bash
python -m src.main adjacent_words.txt налоги доходы 0
```

---

### **Технические заметки**

*   **Кодировка файлов:** Приложение ожидает, что входные файлы будут в кодировке UTF-8.
*   **Разделители слов:** Используется регулярное выражение `\b\w+\b` для токенизации, что означает, что слова состоят из буквенно-цифровых символов, ограниченных границами слов. Знаки препинания и другие символы игнорируются как разделители.
*   **Чувствительность к регистру:** Поиск производится без учёта регистра.
